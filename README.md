# Permit Management System

This project is a comprehensive system for managing council permits.

## Project Roadmap

The project is divided into five phases:

- **Phase 1: The Foundation** (Database & Core Models)
- **Phase 2: The Core Logic** (Services & Basic Endpoints)
- **Phase 3: Security & Staff Management** (The Gatekeeper)
- **Phase 4: The Staff UI** (Thymeleaf MVP)
- **Phase 5: Advanced Features & Customer Portal**
